# Turkish Morphology FST - FST Rules

Bu dizin, HFST için FST kural dosyalarını içerir.

## Dosyalar

- `nominal.xfst` - İsim paradigması kuralları
- `verbal.xfst` - Fiil paradigması kuralları
- `phonology.xfst` - Fonoloji kuralları (ünlü uyumu, ünsüz değişimleri vb.)

## Format

XFST (Xerox Finite State Tool) formatı kullanılmaktadır.
