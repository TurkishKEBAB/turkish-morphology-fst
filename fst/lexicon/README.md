# Lexicon Dosyaları

Bu klasör Türkçe kök sözlüklerini içerir.

## Dosyalar

| Dosya | Açıklama |
|-------|----------|
| `nouns.lexc` | İsim kökleri |
| `verbs.lexc` | Fiil kökleri |
| `adjectives.lexc` | Sıfat kökleri |
| `closed_class.lexc` | Kapalı sınıf kelimeler |

## Format

```lexc
LEXICON Root
ev Noun ;
kitap Noun ;
gel Verb ;
```
