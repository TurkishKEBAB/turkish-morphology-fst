# Türkçe Morfolojik Analizör (HFST)

Berke Özenç danışmanlığında geliştirilen Türkiye Türkçesi morfolojik analizör projesi.

## Yapı

```
fst/
├── lexicon/    # Kök sözlükleri
├── rules/      # Ses kuralları
└── compiled/   # Derlenmiş FST
```
